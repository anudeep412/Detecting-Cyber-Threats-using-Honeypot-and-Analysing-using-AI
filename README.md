# Detecting Cyber Threats using Honeypot and Analysing using AI

## Master's Thesis Project

**Master of Engineering in Cyber Security**
 | University of Limerick

---

## Abstract

This project presents an AI-powered cyber threat hunting framework that combines the Cowrie SSH/Telnet honeypot with Large Language Models (LLMs) to automatically analyse attacker behaviour and map activities to the MITRE ATT&CK framework.

The system captures real-world attacker interactions through a Cowrie honeypot, processes and normalises the generated logs, performs AI-assisted threat analysis using GPT-5 and Mistral models, stores predictions within a SQLite database, and visualises findings through a real-time FastAPI and WebSocket-powered dashboard.

The primary objective is to reduce analyst workload by transforming raw honeypot logs into actionable threat intelligence with confidence scores, ATT&CK mappings, severity ratings, and natural-language explanations.

# Installation

## Prerequisites

* Python 3.10+
* Node.js 18+
* Git
* Cowrie Honeypot
* OpenAI API Key (optional)
* Ollama (for local Mistral inference)

---

## Clone Repository

```bash
git clone https://github.com/anudeep412/Detecting-Cyber-Threats-using-Honeypot-and-Analysing-using-AI.git

cd Detecting-Cyber-Threats-using-Honeypot-and-Analysing-using-AI
```

---

## Backend Setup

```bash
cd BackEnd

python -m venv venv

source venv/bin/activate
```

Windows:

```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run FastAPI:

```bash
python main.py
```

---

## Frontend Setup

```bash
cd FrontEnd

npm install

npm start
```

---

## Cowrie Honeypot Setup

```bash
git clone https://github.com/cowrie/cowrie.git

cd cowrie

python -m venv cowrie-env

source cowrie-env/bin/activate

pip install -r requirements.txt
```

Start Cowrie:

```bash
bin/cowrie start
```

# Workflow

1. Cowrie captures attacker interactions.
2. Logs are normalised and enriched.
3. Rule-based analysis is performed.
4. GPT-5 or Mistral predicts ATT&CK techniques.
5. Results are stored in SQLite.
6. FastAPI exposes analytics APIs.
7. WebSocket streams updates.
8. Dashboard visualises attack intelligence.
